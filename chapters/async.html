Callback

A callback is a function A passed in param of another function B.
Thanks to this, the function B could be use and call the function A without know in advance thus the function B executes dynamic code.

Live Example on NodeJS

ES6 Promise


      // Callback
      step1(value, function(data){
        step2(data, function(data2){
          step3(data2, function(data3){
            step4(data3, function(data4){
              //INSTRUCTIONS
            });
          });
        });
      });

      // Promise
      step1.then(step2) .then(step3) .then(step4)
