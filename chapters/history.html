JavaScript Object

  • In JavaScript, an object is a set of properties
  • A property has a name and a value
  • A property can be a method or an instance variable 

    // Literal Object:
    var myObj = {
      propertyName: “propertyValue”
    };

    // Access property:
    myObj.propertyName;
    myObj[‘propertyName’];
