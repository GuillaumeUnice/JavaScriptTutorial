Callback

A callback is a function A passed in param of another function B.
Thanks to this, the function B could be use and call the function A without know in advance thus the function B executes dynamic code.

Promesse : JavaScript est asynchrone l’attente de données nécessite l’utilisation de callback.
Une promesse au sens JavaScript est une fonctionnalité permettant de retourner des callbacks en cas de succès ou d’erreur.
Il y a un avantage purement syntaxique à pouvoir chaîner les promesses les unes à la suite des autres plutôt qu’utiliser une imbrication de callback.
