XHR JavaScript Object with main goal

  • Send request to server
  • Have got an handler to recup response

JavaScript XMLHttpRequest Send


        // GET request
        var xhr = new XMLHttpRequest();
        xhr.open(‘GET’, ‘http://localhost:3004/series’, true);
        xhr.send(null);
    

        // GET request with query params
        var username = encodeURIComponent(username),
        xhr.open(‘GET’, ‘http://localhost:3004/series?username=’ + username, true);
    

        // POST request with body params
        xhr.open('POST', 'http://localhost:3004/series', true);
        xhr.setRequestHeader("Content-Type", "application/x-www-form-urlencoded");
        xhr.send(title=' + title + '&synopsis=' + synopsis);
