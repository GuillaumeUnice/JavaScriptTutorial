Overview

  • Langage compilé: Code source donné compilateur convertir language machine (ex: C, C++)
  • Langage précompilé: Code source partiellement compilé. Code intermédiaire requiert "une machine virtuelle" pour executer le code (ex: C#, Java)
  • Langage interprété: pas de compilation le code source reste tel quel si on veut executer besoin d'un interpréteur (ex: JS qui dispose de plusieurs moteur/interpreteur selon le navigateur IE/dge => Chakra, SpiderMonkey => Mozilla, V8 Chrome => node

    • Javascript est un langage interprété contrairement au langage compilé il n’y a pas de face de compilation il a juste besoin d’un interpréteur (inclus dans nos navigateurs) et peut donc être compilée à la volée Just In Time pour d’éventuelle amélioration de performance

    Brendan Eich anciennement appelé LiveScript prit le nom de JavaScript choix purement marketing pour surfait sur la vague Java JavaScript sort en décembre 1995,

  • ES 5 (sorti en décembre 2009), la version la plus répandue et utilisée à ce jour
  • ES 6, finalisé en décembre 2014 et dont l'implémentation avait déjà été commencée avant cette date au sein de plusieurs navigateurs.
